Fourier Visualization Tool

This tool visualizes how functions can be approximated using Fourier series.

Given a real-valued function \$f(x)\$ defined on the interval \$[0, L]\$, this tool visualizes the construction of its Fourier sine series, given by:

\$$ \sum_{n=1}^\infty \,b_n \sin{\frac{n \pi x}{L}}\$$

where:

\$$ b_n = \frac{2}{L} \int_0^L \!f(x) \sin{\frac{n \pi x}{L}} \:dx\$$
